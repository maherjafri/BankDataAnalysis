# Bank Client Data Analysis

## Overview
Exploratory Data Analysis (EDA) on 45,211 bank client records from the UCI Bank 
Marketing Dataset to identify credit risk patterns, customer segments, and 
marketing campaign effectiveness.

## Key Findings
- Call duration (r = +0.39) is the strongest predictor of term deposit subscription
- May had peak contact volume (13,766 calls) but only 8.7% conversion vs December at 47%
- Clients with prior successful campaign outcomes convert at ~65% — 6x the base rate
- Blue-collar and unemployed clients carry disproportionately higher default risk

## Tools Used
Python | Pandas | NumPy | Matplotlib | Seaborn

## Dataset
UCI Bank Marketing Dataset — 45,211 records, 17 features
